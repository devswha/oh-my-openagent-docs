## Planning vs Execution

Prometheus plans and does not write code. Metis catches gaps. Momus reviews the plan. Atlas executes approved plans by delegating category+skill tasks.

<Mermaid chart={`
stateDiagram-v2
    [*] --> Interview
    Interview --> Research
    Research --> Clearance
    Clearance --> Interview: unclear
    Clearance --> Plan: clear
    Plan --> Metis
    Metis --> Momus
    Momus --> Plan: rejected
    Momus --> Atlas: okay
    Atlas --> Workers
    Workers --> Verify
    Verify --> Atlas: needs fix
    Verify --> [*]: done
`} />

## Category + Skill

Categories describe the work; skills inject domain expertise.

| Category | Use |
|---|---|
| `visual-engineering` | Frontend, UI/UX, design, animation. |
| `ultrabrain` | Hard logic and architecture. |
| `deep` | Autonomous research and implementation. |
| `quick` | Trivial changes. |
| `writing` | Docs and prose. |
