**Oh My OpenAgent(OmO)** 는 OpenCode 위에 올라가는 멀티 모델 에이전트 하네스입니다. 단순히 프롬프트 몇 개를 모아둔 레이어가 아니라, 의도 파악, 계획, 위임, 실행, 검증, 세션 연속성을 하나의 루프로 묶습니다.

공식 사이트 기준 설치 명령은 아래입니다.

```bash
bunx oh-my-openagent install
```

전환 기간 동안 `oh-my-opencode` 패키지도 같은 버전과 같은 바이너리를 제공합니다. 기존 문서나 자동화가 `bunx oh-my-opencode install`을 사용해도 동작합니다.

## 무엇이 다른가

- **계획과 실행 분리**: Prometheus는 코드를 쓰지 않고 계획만 세웁니다. Atlas는 검증된 계획을 실행합니다.
- **모델명을 직접 고르지 않는 위임**: 작업을 `visual-engineering`, `deep`, `quick` 같은 카테고리로 표현합니다.
- **신뢰하되 검증**: 하위 에이전트 결과를 그대로 믿지 않고, 진단·빌드·테스트·수동 QA로 확인합니다.
- **중단 복구**: Boulder/continuation 계열 훅이 세션 중단과 미완료 todo를 추적합니다.
