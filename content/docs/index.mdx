![Oh My OpenAgent](/images/hero.jpg)

**Oh My OpenAgent (OmO)** is a multi-model agent harness for OpenCode. It is not just a prompt pack: it connects intent detection, planning, delegation, execution, verification, and session continuity.

The public install command is:

```bash
bunx oh-my-openagent install
```

During the rename transition, `oh-my-opencode` remains available at the same version and exposes the same binaries.
