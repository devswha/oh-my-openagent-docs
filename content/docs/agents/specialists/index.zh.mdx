## Specialist Map

| Agent | Specialty | Boundary |
|---|---|---|
| Oracle | Architecture, hard debugging, self-review | Read-only consultant. |
| Librarian | External docs, OSS code, remote repositories | Research-only. |
| Explore | Repo-local search and pattern mapping | Search-only. |
| Multimodal-Looker | Images, screenshots, PDFs, diagrams | Visual extraction, not implementation. |

Use them to reduce guessing before implementation.
