## Category Dispatch

`task(category="...")` routes through Sisyphus-Junior. The category describes the work and OmO selects model/profile instructions.

| Category | Best for |
|---|---|
| `visual-engineering` | Frontend and browser-verified UI work. |
| `ultrabrain` | Hard reasoning and architecture. |
| `deep` | Research plus implementation. |
| `artistry` | Creative or unconventional work. |
| `quick` | Small bounded changes. |
| `writing` | Documentation and prose. |

Skills add domain playbooks on top of categories.
