## 전문 역할

| Agent | 전문 분야 | 경계 |
|---|---|---|
| Oracle | 아키텍처, 어려운 디버깅, 자기 리뷰 | 읽기 전용 consultant. 작은 첫 수정에는 과합니다. |
| Librarian | 외부 문서, OSS 코드, 원격 저장소 | 리서치만 합니다. 편집과 재위임은 안 합니다. |
| Explore | 로컬 코드 검색과 패턴 매핑 | 검색만 합니다. “X가 어디 있나?”에 먼저 부르세요. |
| Multimodal-Looker | 이미지, 스크린샷, PDF, 다이어그램 | 시각 정보 추출용. 구현 워커가 아닙니다. |

## Sisyphus가 쓰는 방식

Sisyphus는 구현 책임을 갖고 있고, 증거가 필요할 때 전문가를 부릅니다. Explore가 로컬 코드를 찾고, Librarian이 외부 문서를 확인하고, Oracle이 위험한 결정을 검토하고, Multimodal-Looker가 스크린샷이나 다이어그램을 읽습니다.

## 가드레일

이 에이전트들은 추측을 줄이려고 존재합니다. 사실, 위험, 다음 조치만 보고합니다. 몰래 범위를 넓히거나 프로덕션 변경을 쓰면 안 됩니다.
