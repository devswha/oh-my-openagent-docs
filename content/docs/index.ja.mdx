![Oh My OpenAgent](/images/hero.jpg)

**Oh My OpenAgent (OmO)** is a batteries-included OpenCode plugin for multi-model agent orchestration. It turns one assistant into a small development team: Sisyphus plans and delegates, specialists research and implement, and verification tools keep the work grounded.

## Quick Start

```bash
bunx oh-my-openagent install
bunx oh-my-openagent doctor
```

Then open OpenCode and type:

```text
ultrawork
```

`oh-my-openagent` と `oh-my-opencode` の両方が公開されています。OmO は `oh-my-openagent` の plugin entry を優先し、`oh-my-opencode` の legacy entry は移行期間中に警告付きでロードされます。
