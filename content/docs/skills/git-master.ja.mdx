Three personas: Commit Architect (atomic splits + style detection), Rebase Surgeon, History Archaeologist.

Triggers: `commit`, `rebase`, `squash`, `who wrote`, `when was X added`. Explicit: `/git-master <command>`.

Defaults: 3+ files → 2+ commits; 5+ → 3+; 10+ → 5+. Auto-detects style from last 30 commits.
