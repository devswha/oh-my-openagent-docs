Install: `bun add -g agent-browser && agent-browser install`.

Switch: `{ "browser_automation_engine": { "provider": "agent-browser" } }`.

Same capabilities as playwright (navigate, snapshots, screenshots, network, scripted interactions), CLI-driven.
