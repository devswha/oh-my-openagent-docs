## 내장 엔지니어링 스킬

| Skill | 용도 |
|---|---|
| `frontend-ui-ux` | 디자인 우선 UI 구현과 시각적 완성도. |
| `playwright` | MCP 기반 브라우저 자동화. |
| `playwright-cli` | MCP 없이 shell 명령으로 브라우저 자동화. |
| `agent-browser` | 이동, snapshot, click, form 입력을 위한 브라우저 명령. |
| `dev-browser` | 반복 웹 개발을 위한 persistent page state. |
| `git-master` | atomic commit, rebase, history search, release-safe git 흐름. |

## 추천 조합

- UI 변경: `category="visual-engineering"`, `load_skills=["frontend-ui-ux", "playwright"]`
- git 작업: `category="quick"`, `load_skills=["git-master"]`
- 웹 QA: 브라우저 스킬에 실제 로컬 URL을 같이 줍니다.
