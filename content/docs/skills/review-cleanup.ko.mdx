## 리뷰 레인

| Skill | 확인하는 것 |
|---|---|
| `review-work` | 목표 부합, 코드 품질, 보안, QA 증거, 문맥 채굴. |
| `ai-slop-remover` | 불필요한 추상화, 중복, 시끄러운 주석 같은 AI-generated smell. |
| `pre-publish-review` | 변경별 deep review, holistic review, Oracle synthesis 기반 릴리스 준비도. |
| `github-triage` | 증거 기반 읽기 전용 issue/PR triage. |
| `work-with-pr` | worktree setup부터 CI/review loop와 merge까지 PR lifecycle 전체. |

## 원칙

리뷰 스킬은 테스트 대체가 아니라 구현 후 보강입니다. 위험과 빠진 증거를 찾아서 구체적인 수정으로 되돌리는 게 목적입니다.
