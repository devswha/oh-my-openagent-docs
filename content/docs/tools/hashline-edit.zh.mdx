title: Hashline Edit
description: Hash-anchored edit tool. Stub for zh.
---

`LINE#ID` hash anchors validate every change. Rejects stale-line edits instead of silently merging them. Bench result: Grok Code Fast 1 went from 6.7% to 68.3% success.

Opt-in: `{ "hashline_edit": true }`. Companion hooks `hashline-read-enhancer` and `hashline-edit-diff-enhancer` auto-enable.
