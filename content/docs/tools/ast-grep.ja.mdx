Two tools: `ast_grep_search`, `ast_grep_replace`. Operates on parse trees, not strings. Handles multi-line calls, nested parens, template strings — regex doesn't.

Pair with LSP rename (identifier-level) for the cleanest refactor. Both used inside `/refactor`.
