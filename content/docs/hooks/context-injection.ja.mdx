`directory-agents-injector` walks AGENTS.md from the file up to project root. `directory-readme-injector` injects neighboring README. `rules-injector` injects `.claude/rules/*.{md,mdc}` by glob. `compaction-context-injector` re-injects load-bearing context after compaction. `context-window-monitor` + `preemptive-compaction` manage the window.
