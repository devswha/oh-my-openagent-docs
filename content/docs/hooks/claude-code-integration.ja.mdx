`claude-code-hooks` runs settings.json `hooks` blocks from `~/.claude/`, `./.claude/`, `./.claude/settings.local.json`.

Disable per-surface: `claude_code.{mcp,commands,skills,agents,hooks,plugins}: false`. Disable specific marketplace plugin: `claude_code.plugins_override`.
