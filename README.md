# oh-my-openagent-docs

Official documentation site for [Oh My OpenAgent](https://github.com/code-yeongyu/oh-my-openagent), a multi-model orchestration harness for OpenCode.

- Framework: Next.js 16 + Fumadocs 16
- Languages: English, 한국어, 中文, 日本語
- Source project: https://github.com/code-yeongyu/oh-my-openagent

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```
